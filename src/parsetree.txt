-> : = x y 2 * 1 +

x = -y * 2 + 1
-> = x y NEG 2 * 1 +

-> ( foo 1 2 3 )

c(4, d(5))
-> ( foo 1 2 ( x b ) ) ( c 4 ( d 5 ) )
